Welcome to nzb360

nzb360 has been designed with love and care to provide the best experience when dealing with the supported usenet/torrent services. If you're unfamiliar with usenet and torrent services, this guide will assist you in learning more about them.

Basic Requirements

To use nzb360 correctly, you will need to have a usenet provider and/or a supported torrent client to get any use out of this application. If you are unfamiliar with usenet, I suggest starting here. If you are already using any of the supported services below, you already have everything you need to get nzb360 up and running.

Supported Services

The following PC/Mac/Linux applications are supported by nzb360. If you don't plan to use any of these applications, nzb360 will not be useful to you.

Usenet Download Clients

  • SABnzbd

    SABnzbd connects to a usenet provider to download files that exist on usenet servers.

  • NZBget

    An alternative to SABnzbd that works better on low-end PCs.

Torrent Clients

  • Deluge

    A torrent download client used to download torrent files.

  • Transmission

    A torrent download client popular among MacOSX users.

  • ÂµTorrent

    A torrent download client maintained by BitTorrent itself.

  • qBittorrent

    Similar to uTorrent, but completely open source.

  • rTorrent/ruTorrent

    A popular torrent client among seedboxes.

TV Show Software

  • Sonarr

    A program designed to manage TV Show content.

  • Sick Beard

    A program designed to manage TV Show content (old, not recommended).

Movie Software

  • Radarr

    A program designed to manage movie content.

Music Software

  • Lidarr

    A program designed to manage music content.

Usenet Indexers

  • Unlimited Newznab-based Indexers

    Usenet indexers are websites that index usenet content. You can add any newznab-based indexer to nzb360 to view this content directly through the app and send it to either SABnzbd or NZBget to download on your PC.

  • Prowlarr

    A program designed to aggregate your indexers and make it easy to search all of them at once. Supports Usenet and Torrent indexers.

  • Jackett

    An alternative to Prowlarr that supports only Torrents.
