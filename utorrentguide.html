µTorrent Setup Guide

This guide will walk you through the steps to get nzb360 connected to µTorrent running on your computer. If you haven't already, download and install µTorrent on your computer.

Step 1: Choose µTorrent as your client

nzb360 supports many different Torrent clients. Make sure to choose µTorrent by tapping the row above primary connection string.

Step 2: Set up Web UI

You must enable the Web UI in µTorrent to allow nzb360 to connect. Follow the steps below to enable within µTorrent.

  • Click on Options -> Preferences.
  • Click on the Web UI tab.
  • Make sure "Enable Web UI" is selected.
  • Add a username/password that you will remember.
  • Enable the "Alternative listening port". Remember this number (usually 8089), you'll need it in the next step.

Step 3: Primary Connection Address

Once you've enabled the web UI, you will need to add your primary connection address into nzb360's Torrent's settings, shown below.

NOTE: nzb360 appends /gui automatically for you. Do not add it to the primary connection string for µTorrent.

To find your primary connection address, follow this guide to find your PRIVATE/LOCAL IP address. It will usually look something like http://192.168.X.X.

Add your IP address to the primary connection string field, along with the alternative listening port from the Web UI settings. Add a colon between the IP address and HTTP port.

µTorrent uses port 8089 by default, so your Primary Connection String should look something like http://192.168.X.X:8089

If you're using a remote address (not a local IP address) then you'll want to use that in your Primary Connection Address.

Once you've entered in your Primary Connection Address, move on to step 4.

Step 4: Adding your password

Fill in the same username/password you added into µTorrent's Web UI settings here.

Step 5: Test your connection

nzb360 allows you to test your connection from the settings screen to see if things will work.

Note: Your mobile device MUST be on the same network as µTorrent (if you're using a local connection, i.e. 192.168.x.x). Make sure your mobile device is on your home wifi.

Press the Test Connection button.

  • If you get a Success you're all set!
  • If you get a Failed to Connect follow the troubleshooting tips below.

Troubleshooting

  • Make sure your Primary Connection String is correct.
  • Double check your username/password.
  • Try logging into µTorrent's Web UI via your browser on your computer by typing in your Primary Connection String exactly. If you can't connect via this method, nzb360 won't work either.
  • Make sure no firewall rules are blocking external access.
  • Make sure you're on the same network and are connected via wifi on your mobile device.
  • Make sure µTorrent is running on your PC and remote access is enabled.
  • Remove /gui in your primary connection string if it exists.
