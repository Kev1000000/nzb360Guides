qBittorrent Setup Guide

This guide will walk you through the steps to get nzb360 connected to qBittorrent running on your computer. If you haven't already, download and install qBittorrent on your computer.

Step 1: Choose qBittorrent as your client

nzb360 supports many different Torrent clients. Make sure to choose qBittorrent by tapping the row above primary connection string.

Step 2: Set up Web UI

You must enable the Web UI in qBittorrent to allow nzb360 to connect. Follow the steps below to enable within qBittorrent.

  • Click on Tools -> Options.
  • Click on the Web UI tab.
  • Make sure "Web User Interface (Remote Control)" is selected.
  • Leave the IP address field as an *.
  • Write down (or remember) the port number. By default, it should be 8082.
  • Add a username/password that you will remember in the authentication section.

Step 3: Primary Connection Address

Once you've enabled the web UI, you will need to add your primary connection address into nzb360's Torrent's settings, shown below.

To find your primary connection address, follow this guide to find your PRIVATE/LOCAL IP address. It will usually look something like http://192.168.X.X.

Add your IP address to the primary connection string field, along with the port from the Web UI settings. Add a colon between the IP address and HTTP port.

qBittorrent uses port 8082 by default, so your Primary Connection String should look something like http://192.168.X.X:8082

If you're using a remote address (not a local IP address) then you'll want to use that in your Primary Connection Address.

Once you've entered in your Primary Connection Address, move on to step 4.

Step 4: Adding your password

Fill in the same username/password you added into qBittorrent's Web UI settings here.

Step 5: Test your connection

nzb360 allows you to test your connection from the settings screen to see if things will work.

Note: Your mobile device MUST be on the same network as qBittorrent (if you're using a local connection, i.e. 192.168.x.x). Make sure your mobile device is on your home wifi.

Press the Test Connection button.

  • If you get a Success you're all set!
  • If you get a Failed to Connect follow the troubleshooting tips below.

Troubleshooting

  • Make sure your Primary Connection String is correct.
  • Double check your username/password.
  • Try logging into qBittorrent's Web UI via your browser on your computer by typing in your Primary Connection String exactly. If you can't connect via this method, nzb360 won't work either.
  • Make sure no firewall rules are blocking external access.
  • Make sure you're on the same network and are connected via wifi on your mobile device.
  • Make sure qBittorrent is running on your PC and remote access is enabled.
