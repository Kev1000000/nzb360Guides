Deluge Setup Guide

This guide will walk you through the steps to get nzb360 connected to Deluge running on your computer. If you haven't already, download and install Deluge on your computer.

Step 1: Choose Deluge as your client

nzb360 supports many different Torrent clients. Make sure to choose Deluge by tapping the row above primary connection string.

Step 2: Set up Web UI in Deluge

nzb360 connects to Deluge's WebUI plugin. You will need to make sure this is downloaded/enabled within your Deluge instance. To verify, follow these steps...

  • Click on the Deluge's settings icon (the wrench/screwdriver)
  • Click on the last tab (WebUI)
  • Make sure "Enable web interface" is selected

Note: Keep an eye on the listening port in this view. You'll need it in the next step.

Step 3: Primary Connection Address

Once you've enabled the WebUI, you will need to add your primary connection address into nzb360's Torrent's settings, shown below.

To find your primary connection address, follow this guide to find your PRIVATE/LOCAL IP address. It will usually look something like http://192.168.X.X.

Add your IP address to the primary connection string field, along with the listening port from the WebUI settings. Add a colon between the IP address and listening port.

Deluge uses port 8112 by default, so your Primary Connection String should look something like http://192.168.X.X:8112

If you're using a remote address (not a local IP address) then you'll want to use that in your Primary Connection Address.

Once you've entered in your Primary Connection Address, move on to step 4.

Step 4: Adding your password

Deluge uses a password (with no username) to allow connections like nzb360. The default password for Deluge is simply deluge

If you'd prefer to change the password, you can by visiting your primary connection string on your computer in your web browser. Once you're logged in, follow these steps...

  • Click on Preferences.
  • Select the Interface category.
  • Fill out the fields under Password.
  • Click Change button.
  • Enter this new password into nzb360.

Step 5: Test your connection

nzb360 allows you to test your connection from the settings screen to see if things will work.

Note: Your mobile device MUST be on the same network as Deluge (if you're using a local connection, i.e. 192.168.x.x). Make sure your mobile device is on your home wifi.

Press the Test Connection button.

  • If you get a Success you're all set!
  • If you get a Failed to Connect follow the troubleshooting tips below.

Troubleshooting

  • Make sure your Primary Connection String is correct.
  • Double check your password.
  • Try logging into Deluge's Web UI via your browser on your computer by typing in your Primary Connection String exactly. If you can't connect via this method, nzb360 won't work either.
  • Make sure no firewall rules are blocking external access.
  • Make sure you're on the same network and are connected via wifi on your mobile device.
  • Make sure Deluge is running on your PC and the WebUI is enabled.
